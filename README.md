## 🎯 Overview

Inferno AI is a production-ready, trauma-informed mental health platform that provides 24/7 support for individuals with PTSD, anxiety, and trauma-related conditions. Built by an autistic founder with lived experience, it combines advanced AI with clinical protocols to deliver accessible, evidence-based care.

### 🏆 Key Features

- **🎤 Voice-First Therapy** - Real-time speech-to-speech processing with AWS Polly Neural TTS
- **🚨 Crisis Detection** - AI-powered intervention with 97% accuracy and immediate professional response
- **🧠 Evidence-Based Protocols** - 7 clinical frameworks with continuous PubMed research integration
- **🛡️ HIPAA Compliance** - Enterprise-grade security with 85-90% compliance (certification in progress)
- **🎓 Professional Training** - Free certification programs for trauma-informed care specialists
- **🎖️ Veterans Community** - Secure, encrypted support networks for military personnel

## 🚀 Quick Start

### Prerequisites
```bash
- Node.js 18+
- PostgreSQL database
- OpenAI API key
- AWS credentials (for Polly voice synthesis)
```

### Installation
```bash
# Clone the repository
git clone https://github.com/EverettNC/Inferno.git
cd Inferno

# Install dependencies
npm install

# Configure environment
cp .env.example .env
# Add your DATABASE_URL, OPENAI_API_KEY, AWS credentials

# Initialize database
npm run db:push

# Start development server
npm run dev
```

### Production Deployment
```bash
# Build for production
npm run build

# Start production server
npm start
```

Visit `http://localhost:5173` to access the platform.

### Tech Stack
